Аутентификация API

Передача JWT токена

Для доступа к защищённым эндпоинтам необходимо передавать JWT в заголовке Authorization: 

Authorization: Bearer {your_jwt_token}

Базовый URL

https://api.authmaster.example.com/v1
Все запросы должны содержать: Content-Type: application/json

Формат ответа

Все ответы возвращаются в формате JSON.

Коды статусов

  • 200 — Успешный запрос
  • 201 — Успешное создание
  • 400 — Неверный запрос
  • 401 — Неавторизованный доступ
  • 403 — Доступ запрещён
  • 404 — Ресурс не найден
  • 500 — Внутренняя ошибка сервера

Эндпоинты аутентификации

POST /auth/register

Регистрация нового пользователя

{
  "email": "user@example.com",
  "password": "securePassword123",
  "name": "Иван Иванов"
}
Успешный ответ (201) 
{
  "success": true,
  "message": "Пользователь успешно зарегистрирован"
}

Ошибки:

  • 400 — Пользователь уже существует
  • 400 — Неверный формат email
  • 400 — Слабый пароль

POST /auth/login

Вход пользователя

{
  "email": "user@example.com",
  "password": "securePassword123"
}
Успешный ответ (200) 
{
  "token": "...",
  "refreshToken": "..."
}

Эндпоинты администратора

GET /admin/users

Получение списка пользователей

Требуются права администратора
Authorization: Bearer {admin_access_token}

Параметры запроса:

  • page — номер страницы (по умолчанию: 1)
  • limit — количество пользователей (по умолчанию: 20)
  • search — поиск по email или имени

Успешный ответ (200):

{
  "success": true,
  "users": [
    {
      "id": "user_123456",
      "email": "user@example.com",
      "name": "Иван Иванов",
      "role": "user",
      "createdAt": "2025-01-15T10:30:00Z",
      "lastLogin": "2025-01-25T09:15:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150,
    "pages": 8
  }
}

PUT /admin/users/{userId}/role

Изменение роли пользователя

Authorization: Bearer {admin_access_token}

Тело запроса:

{
  "role": "admin"
}

Успешный ответ (200):

{
  "success": true,
  "message": "Роль пользователя обновлена",
  "user": {
    "id": "user_123456",
    "email": "user@example.com",
    "role": "admin"
  }
}
